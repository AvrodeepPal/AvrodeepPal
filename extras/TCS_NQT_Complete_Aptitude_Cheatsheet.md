### ‚ö° Quick Shortcuts by Topic

#### **1. SIMPLIFICATION (Very Common - 2-3 Questions)**
**Shortcut:** BODMAS/PEMDAS - Bracket ‚Üí Order ‚Üí Division/Multiplication ‚Üí Addition/Subtraction

**Example:**
```
Q: 24 √∑ 6 √ó 3 + 2 - 1 = ?
Method: 4 √ó 3 + 2 - 1 = 12 + 2 - 1 = 13 ‚úì
```

**High-Prediction Topics:**
- Mixed operations with fractions
- Decimal simplifications
- Percentage in formulas

**Tricks:**
- Work left to right for same priority operations
- Convert decimals to fractions for easier calculation
- Use approximation when answer choices are far apart

**Time Goal:** 30-60 seconds per question

#### **2. TIME & WORK (Frequently Asked - 2-3 Questions)**

**Main Formula:** Work = Rate √ó Time OR W = 1/(Time to complete)

**When A & B work together:**
```
Combined Rate = 1/a + 1/b
Time taken together = (a √ó b)/(a + b)
```

**High-Prediction Variations:**
- Both work together then one leaves
- Work completed at different rates
- Capacity/efficiency changes

**Example:**
```
Q: A can do work in 10 days, B in 15 days. How long together?
Answer: (10 √ó 15)/(10 + 15) = 150/25 = 6 days ‚úì

Quick Trick: If A:B ratio is 10:15 = 2:3
Combined work in 1 day = 1/10 + 1/15 = (3+2)/30 = 5/30 = 1/6
So 6 days total ‚úì
```

**Common Traps:**
- Question says "remaining work" (read carefully)
- Efficiency changes mid-problem
- Different time units (hours vs days)

**Time Goal:** 1-1.5 minutes per question

#### **3. PIPES & CISTERNS (Same as Time & Work - 2 Questions)**

**Key Difference:** Inlet(+), Outlet(-), Both(-) when draining

**Formula:**
```
Part filled = Inlet rate √ó Time - Outlet rate √ó Time
If net = 1/8 per hour, time = 8 hours
```

**High-Prediction Scenarios:**
- Multiple inlets/outlets
- Different sized pipes
- Pipe opened/closed mid-process

**Trick:** Convert to LCM-based capacity method

**Time Goal:** 1.5 minutes per question

#### **4. PERCENTAGE (Most Frequent - 2-3 Questions)**

**Convert to Fractions for Speed:**
```
12.5% = 1/8
16.67% = 1/6
20% = 1/5
25% = 1/4
33.33% = 1/3
50% = 1/2
66.67% = 2/3
```

**Successive Percentage Change:**
```
If increase a% then b%:
Final = Original √ó (1 + a/100) √ó (1 + b/100)
Quick: a + b + ab/100 percentage change
```

**High-Prediction Questions:**
- Percentage increase/decrease calculation
- Profit/loss percentage with cost price
- Population growth problems
- Discount chain problems

**Example:**
```
Q: 25% discount on marked price of ‚Çπ1000, then 20% more discount. Final price?
Wrong: 1000 - 250 - 200 = 550
Correct: 1000 √ó 0.75 √ó 0.80 = 600 ‚úì

Quick: 25 + 20 - (25√ó20/100) = 40% total discount
```

**Time Goal:** 60-90 seconds

#### **5. PROFIT & LOSS (Common - 1-2 Questions)**

**Basic Formulas:**
```
Profit% = (SP - CP)/CP √ó 100
Loss% = (CP - SP)/CP √ó 100
SP = CP + Profit (or SP = CP - Loss)
```

**High-Prediction Scenarios:**
- Cost price finding from profit%
- Multiple item transactions
- Marked price vs selling price

**Trick:** When two items sold at same price
```
If CP1=a, CP2=b, both sold at same price P:
Profit% on item1 and loss% on item2
Result typically loss (use harmonic mean)
```

**Time Goal:** 60-90 seconds

#### **6. SIMPLE & COMPOUND INTEREST (1-2 Questions)**

**Simple Interest:** SI = (P √ó R √ó T)/100

**Compound Interest:** A = P(1 + R/100)^T, CI = A - P

**Fast Track:**
```
CI for 2 years = P √ó R √ó (200 + R)/10000

Example: P=1000, R=10%, T=2
CI = 1000 √ó 10 √ó 210/10000 = 210 ‚úì
```

**High-Prediction Questions:**
- Difference between CI & SI
- Quarterly/Half-yearly compounding
- Finding rate or time

**Approximation Trick (Small %):**
```
For small R and small T:
CI ‚âà SI + (SI √ó R √ó T)/(100 √ó 100)
```

**Time Goal:** 90 seconds

#### **7. RATIO & PROPORTION (1-2 Questions)**

**When A:B:C = x:y:z**
```
A = kx, B = ky, C = kz (where k is multiplier)
Find k using given condition, then find values
```

**Allegation Method (for mixtures):**
```
Cost price of unit quantity:
(Q1 √ó CP1 + Q2 √ó CP2)/(Q1 + Q2) = Mean CP
```

**High-Prediction Questions:**
- Three-quantity ratio problems
- Mixture problems with multiple items
- Partition problems

**Time Goal:** 60-90 seconds

#### **8. SPEED, TIME & DISTANCE (1-2 Questions)**

**Basic:** Speed = Distance/Time

**Unit Conversion:**
```
km/hr to m/s: multiply by 5/18
m/s to km/hr: multiply by 18/5
```

**Relative Speed:**
```
Same direction: |S1 - S2|
Opposite direction: S1 + S2
Time to meet = Distance/(Relative Speed)
```

**High-Prediction Questions:**
- Two objects moving toward/away each other
- Boat in river (upstream/downstream)
- Average speed calculations

**Boat Trick:**
```
Upstream: (Speed in still water - River flow)
Downstream: (Speed in still water + River flow)

Average = (2 √ó Upstream √ó Downstream)/(Upstream + Downstream)
```

**Time Goal:** 90 seconds

#### **9. PROBABILITY (1 Question - Frequently Asked)**

**Basic:** P(E) = Favorable outcomes / Total outcomes

**Card Problems (52 cards, 4 suits):**
```
P(specific card) = 1/52
P(red card) = 26/52 = 1/2
P(face card) = 12/52 = 3/13
```

**Dice (1 die):**
```
Total outcomes = 6
P(even) = 3/6 = 1/2
P(number > 3) = 3/6 = 1/2
```

**High-Prediction Topics:**
- Dependent vs independent events
- P(A and B) = P(A) √ó P(B)
- P(A or B) = P(A) + P(B) - P(A and B)

**Complement:** P(not E) = 1 - P(E)

**Time Goal:** 60 seconds

#### **10. NUMBER SYSTEM (1-2 Questions)**

**Divisibility Rules:**
```
Div by 2: Last digit even
Div by 3: Sum of digits divisible by 3
Div by 4: Last 2 digits divisible by 4
Div by 5: Last digit 0 or 5
Div by 6: Divisible by both 2 and 3
Div by 8: Last 3 digits divisible by 8
Div by 9: Sum of digits divisible by 9
Div by 11: (Sum of odd place digits - Sum of even place digits) divisible by 11
```

**HCF/GCF:** Highest common factor (use prime factorization)

**LCM:** Least common multiple (product of highest powers of all prime factors)

**High-Prediction Questions:**
- Largest number dividing given numbers
- Smallest number divisible by given numbers
- Remainder when large numbers divided

**Remainder Trick:** Use modular arithmetic, break large numbers

**Time Goal:** 60-90 seconds

#### **6. VENN DIAGRAMS (Common - 1-2 Questions)**

**Formula for 2 sets:**

## üìù FOUNDATION VERBAL ABILITY

### ‚ö° Quick Shortcuts by Topic

#### **1. SENTENCE CORRECTION / SPOTTING ERRORS (1-2 Questions)**

**Grammar Rules - Master These 5:**

**Rule 1: Subject-Verb Agreement**
```
Each/Every/Either/Neither ‚Üí Singular verb
"Each student has..." (NOT "have")
"Either option is..." (NOT "are")

Plural subjects ‚Üí Plural verb
"Students have..." (NOT "has")
```

**Rule 2: Tense Consistency**
```
If past tense started, maintain it
"He went to market and bought groceries" ‚úì
"He goes to market and bought groceries" ‚úó

Reported speech uses past
"He said that he would help" ‚úì
"He said that he will help" ‚úó (informal speech only)
```

**Rule 3: Pronoun Reference**
```
Pronoun must refer to nearest appropriate noun
Unclear: "When John called Mike, he was tired"
Clear: "When John called Mike, John was tired" OR "...Mike was tired"

Avoid: it, this, that without clear reference
```

**Rule 4: Preposition Usage**
```
Correct: in, on, at, from, to, with, about, of, for, by
"Good at math" (NOT "good in math")
"Interested in science" (NOT "interested about")
"Different from" (NOT "different than")
```

**Rule 5: Articles (a, an, the)**
```
"a" before consonant sound: a book, a university
"an" before vowel sound: an apple, an honor

"the" for specific: The teacher (specific one)
No article for general: Teachers are important
```

**High-Prediction Error Types:**
- Subject-verb mismatch (most common)
- Wrong preposition
- Tense inconsistency
- Article mistakes

**Quick Method:**
- Read sentence aloud mentally
- Check subject-verb first
- Check tense consistency
- Check prepositions

**Time Goal:** 45-60 seconds per question

#### **4. PARA JUMBLES (1-2 Questions)**

**Steps:**
1. Read all sentences first
2. Find OPENING sentence (introduces topic, no pronouns/connectors)
3. Find CLOSING sentence (concludes, summarizes)
4. Connect middle sentences using pronouns & connectors

**Key Indicators:**
```
Opening: Topic introduction, no "he/she/it/they"
Middle: Pronouns (he, she, it), connectors (however, moreover)
Closing: Conclusion, "thus", "therefore", "hence"
```

**Common Opening Signals:**
- Introduces new topic/person
- No pronouns referring to previous statement
- Defines or explains something new

**High-Prediction Pattern:**
- Usually 4 sentences (ABCD)
- One clear opening, rest need ordering
- Common orders: ABCD, ACBD, CADB

**Example:**
```
A: "The company decided to expand globally"
B: "This strategy was risky but profitable"
C: "CEO announced the new expansion plan"
D: "However, competitors had similar plans"

Order: C (introduces plan) ‚Üí A (explains decision) ‚Üí B (evaluates) ‚Üí D (contrast)
```

**Time Goal:** 60-90 seconds

### Advanced Topics - High Prediction:

#### **1. PERMUTATION & COMBINATION (2-3 Questions)**

**Formulas:**
```
nPr = n!/(n-r)! [order matters - arrangements]
nCr = n!/[r!(n-r)!] [order doesn't matter - selections]

nCr = nC(n-r) [quick calculation]
nPr = nCr √ó r! [relationship between both]
```

**High-Prediction Scenarios:**
- Committee formation with restrictions
- Arrangement of letters/words
- Selection with/without repetition

**Example:**
```
Q: In how many ways can 5 people sit in a line?
Answer: 5P5 = 5! = 120 ‚úì

Q: How many ways to select 3 from 5 people?
Answer: 5C3 = 10 ‚úì

Q: Select 3 from 5 people AND arrange them in 2 seats?
Answer: 5C3 √ó 3P2 = 10 √ó 6 = 60 ‚úì
```

**Circular Arrangement:**
```
Linear: n! arrangements
Circular: (n-1)! arrangements
Necklace (both directions): (n-1)!/2

Reason: Fix one position to eliminate rotations
```

**Time Goal:** 2-3 minutes for 2-3 questions

---

#### **2. PROBABILITY WITH MULTIPLE EVENTS (2-3 Questions)**

**Advanced Scenarios:**

**Independent Events:**
```
P(A and B) = P(A) √ó P(B)

Example: Coin + Dice
P(Heads AND 4) = 1/2 √ó 1/6 = 1/12
```

**Dependent Events:**
```
P(A then B | A happened) = different probability
P(B|A) = P(A and B)/P(A)

Example: Drawing 2 cards without replacement
1st card red: 26/52
2nd card red (1st was red): 25/51
Both red: 26/52 √ó 25/51
```

**Mutually Exclusive:**
```
P(A or B) = P(A) + P(B) [if cannot happen together]

Example: Rolling even or odd on one die
P(Even or Odd) = 3/6 + 3/6 = 1 ‚úì
```

**Not Mutually Exclusive:**
```
P(A or B) = P(A) + P(B) - P(A and B)

Example: Drawing red or face card
P = 26/52 + 12/52 - 6/52 = 32/52 = 8/13
```

**Conditional Probability:**
```
P(A|B) = P(A ‚à© B)/P(B)

Example: P(sick|positive test) = needs formula application
```

**Time Goal:** 2-3 minutes total

---

#### **3. DATA INTERPRETATION (3-4 Questions)**

**Question Types:**
- Pie chart ‚Üí percentage/angle calculations
- Bar chart ‚Üí comparison & calculation
- Table ‚Üí sorting & filtering
- Line graph ‚Üí trend analysis

**High-Prediction Calculations:**
- Percentage increase/decrease
- Average calculations
- Ratio comparisons
- Profit/loss percentages

**Trick:**
- Don't calculate exact value, estimate
- Use approximation to eliminate options
- Check units (per 1000, in percentage, etc.)

**Example:**
```
Pie chart shows 45% profit in 2023
If total profit = ‚Çπ50 Crore
What is 2023 profit?
= 50 √ó 45/100 = ‚Çπ22.5 Crore

Quick: If options are far apart (‚Çπ20Cr, ‚Çπ50Cr, ‚Çπ80Cr)
Estimation: ~45% of 50 = ~22.5, close to ‚Çπ20Cr ‚úì
```

**Time Goal:** 1-1.5 minutes per question
